API Reference

Base URL: https://token.route.worthwolf.top

POST /v1/chat/completions

Send messages to AI models with automatic provider routing. 

{
  "messages": [
    {"role": "user", "content": "What is machine learning?"}
  ],
  "provider": "deepseek",    // optional
  "model": "deepseek-chat",  // optional
  "temperature": 0.7

POST /v1/mask

Automatically detect and mask Personally Identifiable Information. 

{
  "text": "Contact Zhang Wei at 13812345678 or zhang@example.com"
}
// Response:
{
  "success": true,
  "masked": "Contact Zhang Wei at [手机号] or [邮箱]",
  "detected": [
    {"type": "PHONE_CN", "value": "13812345678"},
    {"type": "EMAIL", "value": "zhang@example.com"}
  ]
}

Perform local computations on sensitive data — FREE

// Response:
{
  "success": true,

Code Examples

Integrate TokenRouter into your applications

🐍
Python 
import requests

API_KEY = "YOUR_API_KEY"
BASE = "https://token.route.worthwolf.top"

headers = {"Authorization": f"Bearer {API_KEY}"}

# Chat API
def chat(message):
    return requests.post(
        f"{BASE}/v1/chat/completions",
        headers=headers,
        json={"messages": [{"role": "user", "content": message}]}
    ).json()

# PII Masking
def mask(text):
    return requests.post(
        f"{BASE}/v1/mask",
        headers=headers,
        json={"text": text}
    ).json()

# Privacy Compute (FREE)
def stats(data):
    return requests.post(
        f"{BASE}/v1/privacy/compute",
        headers=headers,
        json={"task": "Calculate", "data": data, "operations": ["sum", "average"]}
    ).json()
